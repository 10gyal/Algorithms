
*Input File Requirement* 
THE INPUT FILE MUST BE ONLY NUMBERS SEPARATED BY COMMAS OR "\n". See input1.txt for reference

*Directions for running the program*
1. Run <<python3 main.py>> in the terminal
2. Enter the name of input file
3. Enter the rank of the ith smallest element you want to find

Creates 3 new files for per input file:
* random_input_file.txt:
    Displays result obtained using Randomized Selection and time taken
* deter_input_file.txt:
    Displays result obtained using Deterministic selection and the time taken
* result_input_file.txt
    Displays the correctness of the algorithms and also shows the size of the input file (no. of elements)
